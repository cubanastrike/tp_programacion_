
# 💻 Portfolio Web - Matías Barú

## 📌 Descripción
Este proyecto consiste en un portfolio web desarrollado únicamente con HTML. 
Su objetivo es mostrar información personal, tecnologías aprendidas y un formulario de contacto.

---

## 📂 Contenido
El sitio web incluye:

- 🏠 Página principal (Inicio)
- 🙋‍♂️ Sección "Sobre mí"
- 📁 Sección de proyectos
- 🧠 Tabla de tecnologías aprendidas
- 📩 Formulario de contacto
- 🔗 Enlaces de navegación internos

---

## 🛠️ Tecnologías utilizadas

- HTML5

---

## 🎯 Objetivo del proyecto
Aplicar los conceptos básicos de HTML, incluyendo:

- Uso de etiquetas semánticas (`header`, `nav`, `main`, `section`, `footer`)
- Creación de listas
- Uso de tablas
- Formularios con validación
- Navegación con anclas
